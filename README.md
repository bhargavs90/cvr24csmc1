# How to Submit Your Task Sheet
# Follow these steps carefully. Read each step fully before clicking.

### Step 1 — Create your GitHub account
- Go to **github.com** → click **Sign up**
- Use your **roll number** as username → example: `24csmc66xx`(xx- last 2 character of your Rno)
- Verify your email

### Step 2 — Fork this repository
- Come back to this page
- Click the **Fork** button (top right corner of this page)
- Click **Create fork**
- ✅ You now have your own copy of this repository

### Step 3 — Upload your file in YOUR fork
- You are now in **your fork** (check URL — it shows YOUR username)
- Click **Add file** → **Upload files**
- Upload your task sheet
- **Name your file as your roll number** → example: `24csmc66xx.pdf`
- formats accepted — pdf, docx

### Step 4 — Commit the upload
- Scroll down → in the commit message box type:
  `Upload task sheet - 24csmc66xx.pdf` and mention your complete rno (for reference)
- Click **Commit changes**


## ❗ Important Rules
- File name must be your roll number — wrong name = not counted
- One file per student only
